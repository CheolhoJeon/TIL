# Hello, World

* 코틀린은 `//`, `/**/` 주석 스타일을 지원함
*   코틀린 프로그램의 엔트리 포인트(entry point)는 `main()` 함수임

    ```kotlin
    fun main() {
    	println("Hello, world!")
    }
    /* Output:
    Hello, world!
    */
    ```
* 코틀린에서는 표현식(expression)이나 구문(statement)의 끝에 `;`을 사용할 필요가 없음
* 세미콜론은 오직 한 줄 내에서 여러 표현식 혹은 구문을 구분하기 위해서 사용됨
